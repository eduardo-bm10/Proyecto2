ESPECIFICACIONES DEL PROGRAMA

ESTE PROGRAMA FUE REALIZADO CON PYTHON 3.8.3

LAS BIBLIOTECAS UTILIZADAS FUERON:
- TKINTER 
- GLOB
- TIME
- RANDOM
- THREADING
- OS

***IMPORTANTE***

ESTE PROGRAMA REQUIERE TENER INSTALADO LA BIBLIOTECA PYGAME

Favor instalarla desde el CMD de Windows como pip install pygame, de lo contrario no se podra ejecutar el programa.


Desplegar las autodocumentación de los módulos: mi_auto_doc()
	
Existe una función llamada mi_auto_doc() que al invocarla, realiza un despliegue de toda la documentación de los módulos
principales mediante la función print("nombredelmódulo".__doc__), esto para cada uno de ellos.
